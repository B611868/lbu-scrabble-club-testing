# lbu-scrabble-club-testing

This repo contains implementations of classes for a Scrabble Club member, and for the Leaderboard.

There is a basic test class for each. The mission is to increase the test coverage for each. It should
be reasonably easy to get coverage for `Member` to 100%. Some magic would be needed to test the display
methods in `Leaderboard`, but even without that, 70% or more is possible.

The classes are believed to be correct. Probably.
